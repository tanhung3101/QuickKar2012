System Requirements

To use QuickKar 2012, your system must meet those requirements:
  • CPU: Pentium 3 or higher
  • RAM: 1GB or more
  • HDD: 512MB free space or more
  • OS: Windows 2000/XP/Vista/7/8 32/64 bit with approprite JRE installed. Linux (Fedora 14+ or Ubuntu 10.04+) with JRE installed.
  • JRE: 1.6 or higher

    •

    System Preparation

    If your system has no JRE installed yet, please download and install it at:     http://java.com/en/download/index.jsp
    Clean your hard disk to have at least 512 MB free space. Additional free space may be required to store database and other user data.

    Install QuickKar 2012

    QuickKar 2012 is a Java-based desktop application, so all you need to do is to download or copy the .jar file to your desire destination. Then double click on it.
    On most systems with appropriate installed JRE, the program will run automatically.
    If it does not run automatically, or is opened by an archive manager, please run the following command in your CMD:
    java -jar [Jar file Name]

    To run QuickKar 2012 from the submitted zip file, do the following:
  • Extract the zip file into a folder, e.g. "Assignment01"
  • Open Netbeans, press Ctrl + Shift + O and browse to the "Assignment01", inside that folder, there is a Netbeans project folder named "QuickKar2012". Click to select "QuickKar2012" and click Open Project.

  • Right-click on the project QuickKar2012 and select Run

  • The program will be compiled and executed
